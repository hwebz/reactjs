Pro React Book

The book teaches you how to use React completely, and learn best practices for creating interfaces in a composable way.

Additionaly, It also covers tools and libraries in the React ecosystem (such as React Router and Flux/Redux architectures). Each topic is covered clearly and concisely and is packed with the details you need to learn to be truly effective.
